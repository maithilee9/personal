Instructions

Hello everyone, this is a calculator where you can add, subtract, multiply, divide and get a table of any number.

In the navigation bar bar you can see 5 options. If you go to anyone of them you will see two boxes(from add to division) you can write two numbers and you will get your answer.

Pls refresh the page after every calculation.

In Tables, you will get two tables. The first table you will give the number you want to get the table of. By the way the second table you will write the number you want to get the table till where. (Example) if i write 5 in the first table and 3 in the first table then I will get:

8 x 1 = 8

8 x 2 = 16

8 x 3 = 24
